# 2048游戏重写版本

## 简介
这是一个重写的2048游戏，采用简约风格设计，具有以下特性：

## 主要特性

### 🎮 游戏功能
- **多难度选择**: 支持简单(3x3)、普通(4x4)、困难(5x5)三种模式
- **撤销功能**: 支持撤销最多5步操作
- **暂停/继续**: 可以暂停和继续游戏
- **实时统计**: 显示分数、最佳分数、游戏时间和移动次数

### 🎨 设计风格
- **简约现代**: 采用现代简约的UI设计
- **响应式布局**: 适配桌面和移动设备
- **流畅动画**: 方块移动和合并动画
- **渐变背景**: 美观的渐变色背景

### 💾 数据存储
- **格式统一**: 使用统一的游戏记录格式
- **自动保存**: 游戏结束自动保存记录
- **双重存储**: 支持IndexedDB和localStorage备用存储

### 🎯 控制方式
- **键盘控制**: WASD键或方向键
- **触摸控制**: 支持手机滑动操作
- **按钮控制**: 鼠标点击按钮操作

## 数据格式

游戏记录采用以下统一格式：
```javascript
{
  "id": "uuid-v4-string",           // 唯一记录ID
  "userId": "user_10086",           // 用户ID
  "gameType": "game2048",           // 游戏类型
  "timestamp": 1703688000000,       // 游戏结束时间
  "startTime": 1703687000000,       // 开始时间
  "result": "won",                  // 游戏结果: won, lost, ongoing, abandoned
  "score": 1200,                    // 最终分数
  "duration": 45,                   // 游戏耗时(秒)
  "moves": 12,                      // 移动次数
  "difficulty": "normal",           // 难度: easy, normal, hard, custom
  "meta": {
    "gridSize": "4x4",              // 网格大小
    "maxTile": 2048,                // 最高方块
    "winTarget": 2048               // 胜利目标
  }
}
```

## 文件结构

- `subpages/game2048_new.html` - 游戏主页面
- `css/game2048_new.css` - 样式文件
- `js/game2048_new.js` - 游戏逻辑文件

## 使用方法

1. 打开 `game2048_new.html` 文件
2. 选择难度级别
3. 使用WASD键或方向键移动方块
4. 尝试合并相同数字的方块达到目标数值
5. 游戏结束后可查看详细统计信息
